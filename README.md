# overview

a minimal desktop running **Void Linux** with a suckless-based workflow.

## system

| component            | software         |
| -------------------- | ---------------- |
| operating system     | void linux       |
| window manager       | suckless `dwm`   |
| terminal emulator    | suckless `st`    |
| application launcher | suckless `dmenu` |
| shell                | `fish`           |

## applied patches

* **fullgaps**
